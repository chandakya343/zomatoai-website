Why have both Active and Permanent memory?

Because food history is low-signal information.

‚ùå Putting ALL history in context

  • 100+ detailed order records
  • Pollutes the context window
  • Slows down processing
  • AI gets confused by noise
  • Most orders aren't relevant

Result: Slower, worse recommendations

‚úÖ Smart Memory System

Active Memory (Last 20)

Recent, detailed feedback

"Loved Chicken Biryani - spices perfect (5‚òÖ)"

Permanent Memory (Consolidated)

Extracted patterns & insights

"Allergic to peanuts, prefers medium-high spice, loves biryani"

Result: Fast, highly relevant recommendations

Key Insight: We don't need to remember every single order. We need to remember what MATTERS ‚Äî your preferences, patterns, and recent context.
